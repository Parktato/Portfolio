Tempo Modulation Calculator

I built this application so a musician could quickly calculate a new tempo based on an existing tempo and rhythmic value. I used React in the app to practice building an interactive component and modifying state data with a form.

Practice Routine Generator

I made this app to automate a process which I was doing by hand: randomly selecting exercises to build the technique section of my practice routine. The app is made with React, and javascript shuffles existing arrays of exercises. A particular challenge was changing not only the order of the exercises, but also what kind of exercise for a specific category, and sometimes a key for that exercise. Currently hardcoded with my personal preferred exercises, I would love to expand this to allow users to enter their own sets of exercises and save specific routines.

Informational Page

This project outlines a basic introduction to music theory concepts. It allowed me to practice navigation elements, organizing large groups of text, and communicating technical information using <code> snippets.
